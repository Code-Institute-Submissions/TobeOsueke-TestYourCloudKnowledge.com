Test Your Cloud Knowledge: A learning and development plateform

Purpose:

This is a learning platform that allows you test your knowledge in Cloud technologies like AWS and Azure in timed test format.

How to use this plateform:
  1. First register your username here.
  2. Then select what cloud provider you will like to test your knoledge on.
  3. Click start button when you are ready to begin. Note that the test is timed. Meaning the test will end automatically when the time elaps.
  4. Click the submit button when you are done.
  5. Then your result will be display along side the answers to the questions. So you can learn the correct answer if you missed any.
